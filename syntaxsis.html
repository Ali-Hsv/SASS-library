Lets start learn Syntax!

Text:

Change Color

class="text-[color]"

Example:

class="text-white"

White Text

class="text-green"

Green Text

class="text-purple"

Purple Text

class="text-orange"

Orange Text

class="text-red"

Red Text

Change Color Light & Dark vers.

class="text-[color]-[l & d]-[num]"

All Size: from 1 thourh 9

Dont work at White and Black!

Example:

class="text-yellow-light-4"

Yellow Light 40%

class="text-green-dark-7"

Green Dark 70%

class="text-purple-light-8"

Purple Light 80%

class="text-orange-dark-1"

Orange Dark 10%

Change Hover

class="text-[color]-hover"
class="text-[color]-[l & d]-[num]-hover"

Example:

class="text-yellow-light-7-hover"

Hover Me!

class="text-green-dark-7-hover"

Hover Me!

class="text-purple-hover"

Hover Me!

class="text-orange-hover"

Hover Me!

Change Font Family

class="font-styl-[num]"

All Property: 1 - $font-roboto, 2 - $font-edu-hand, 3 - $font-edu-hand-pre, 4 - $font-oswald, 5 - $font-nunito

Example:

class="font-styl-1"

Roboto

class="font-styl-2"

Edu SA Hand

class="font-styl-3"

Edu VIC WA NT Hand Pre

class="font-styl-4"

Oswald

class="font-styl-5"

Nunito

Change Size

class="font-[size]"

All Size: sm, md, lg, xl, xxl

Example:

class="font-sm"

Small Size

class="font-md"

Medium Size

class="font-lg"

Large Size

class="font-xl"

Extra Large Size

class="font-xxl"

Extra Extra Large Size

Change Decoration

class="text-dec-[property]"

All Property: n - none, u - underline, o - overline, l - line-through

Example:

Deafult
class="text-dec-n"
None
class="text-dec-u"

Underline

class="text-dec-o"

Overline

class="text-dec-l"

Line Through

Change Weight

class="font-w-[property]"

All Property: n - none, b - bold, l - lighter, br - bolder

Example:

class="font-w-n"

None

class="font-w-b"

Bold

class="font-w-l"

Lighter

class="font-w-br"

Bolder

Change Align

class="text-align-[property]"

All Property: l - left, r - right, c - center, j - justify, s - start, e - end, init - inherit, inal - initial, u - unset
